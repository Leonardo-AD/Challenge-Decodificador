<h1 align='center'>Challenge-Decodificador</h1>

<p>Challenge-Decodificador is a project for Oracle ONE class 5 made in partnership with Alura. In this application we can encrypt and decrypt messages. In this app I am using HTML, CSS, JavaScript, Sweet Alert and Phosphor Icons.</p>

## Features Developed

- [x] Encrypt and decrypt messages
- [x] Light and dark mode
- [x] Responsivity
